# 🚀 Быстрый старт — за 5 минут

Хочешь создать свой первый лендинг Kaiten? Начни отсюда.

## Шаг 1: Подготовка (2 минуты)

Определи для себя:

1. **Кто целевая аудитория?**
   - Пример: "Менеджеры проектов в веб-агентствах"

2. **Какие 2-3 основные проблемы этой аудитории?**
   - Пример: "Много проектов одновременно", "Нужна видимость для клиентов", "Интеграции с инструментами"

3. **Какие 2-3 компании мог бы упомянуть как кейсы?**
   - Пример: "Яндекс.Маркет", "Авито", "лю бая крупная компания"

## Шаг 2: Готовый промпт (1 минута)

Скопируй этот текст и подставь свои данные:

```
Создай полный лендинг Kaiten для [СФЕРА]

Целевая аудитория:
[Опиши кто будет читать этот лендинг]

Ключевые проблемы:
- [Проблема 1]
- [Проблема 2]
- [Проблема 3]

На выходе жду готовую HTML-верстку, SEO-данные и чек-лист.
```

## Шаг 3: Отправляй Claude (2 минуты)

Скопируй свой промпт и отправь Claude в этот репозиторий.

Claude прочитает:
- Правила (CLAUDE.md)
- Методологию (kaiten-landing-factory/SKILL.md)
- Редполитику (kaiten-landing-factory/references/brand_policy/)
- Примеры (kaiten-landing-factory/references/landing_tz_examples/)

И создаст тебе полный лендинг.

## Что ты получишь

В папке `outputs/` появится твой лендинг:

```
outputs/kaiten-for-[sfera]/
├── landing.html    # Открой в браузере — готов к публикации
├── seo.json       # SEO-данные для разработчика
├── checklist.md   # Контрольный список
└── references.md  # Использованные источники
```

## Примеры готовых промптов

### Вариант 1 (быстрый)
```
Создай лендинг Kaiten для веб-агентств
```

### Вариант 2 (с деталями)
```
Создай лендинг Kaiten для веб-агентств

Целевая аудитория:
Проект-менеджеры в агентствах (5-100 человек), управляют множеством проектов для разных клиентов

Ключевые проблемы:
- Много проектов одновременно
- Нужна видимость для клиентов, но безопасно
- Сложно контролировать время на каждый проект

На выходе жду готовую HTML-верстку, SEO-данные и чек-лист.
```

## Файлы, которые помогут

- **README.md** — общая информация о проекте
- **CLAUDE.md** — правила для Claude (обязательно прочитай!)
- **PIPELINE.md** — полный процесс от идеи до публикации
- **PROMPTS.md** — готовые примеры промптов
- **kaiten-landing-factory/SKILL.md** — подробное руководство
