# Kaiten для канбан-команд

## Описание

Лендинг для команд, которые используют канбан-методологию управления проектами. Целевая аудитория — agile-команды, разработчики, продакты и менеджеры, которые хотят видеть весь процесс на одной доске.

## Целевая аудитория

- Agile-команды (5-30 человек)
- Технические лиды и скрам-мастера
- Продуктовые менеджеры
- Компании, использующие канбан или lean-метод

## Ключевые месседжи

- Вся работа в одном месте
- Видимость для всех
- Контроль над потоком задач
- Быстрая реакция на изменения

## Структура лендинга

1. **Hero** — Канбан-доска, которая работает
2. **Проблема** — Что мешает командам сейчас
3. **Решение** — Как Kaiten это решает
4. **Фичи** — Основные возможности для канбан-команд
5. **Кейсы** — Реальные примеры (компании, которые используют Kaiten)
6. **FAQ** — Часто спрашиваемые вопросы
7. **CTA** — Призыв к действию
