# Kaiten Landing Factory Skill — полный пакет

В архиве:

- `SKILL.md` — скилл для агента по созданию готовых лендингов Kaiten.
- `Kaiten_Landing_Factory_SKILL.docx` — версия скилла для чтения и согласования.
- `references/brand_policy/` — редполитика Kaiten.
- `references/landing_tz_examples/` — ТЗ-референсы лендингов.
- `references/links/kaiten_public_references.md` — ссылки на публичные референсы, базу знаний и кейсы.

В этой версии добавлено правило: на публичных страницах заменять англицизмы русскими словами или раскрывать их в скобках. Исключения — названия сервисов, товарные знаки и термины без точного русского аналога, например канбан и скрам.
