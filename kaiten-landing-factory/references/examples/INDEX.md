# Индекс примеров лендингов и копи

Используй эти примеры как эталон для структуры и тона текстов при создании новых лендингов.

## 📋 Примеры по сферам

### 🔧 Для производств и логистики
- `Кайтен для производств.docx` — управление производственными процессами
- `kaiten-construction-engineering-landing.docx` — строительство и инжиниринг

### 👥 Для команд и процессов
- `Кайтен для канбан-команд.docx` — agile-команды с канбан-методом
- `Кайтен для скрам.docx` — scrum-команды (если есть)

### 📰 Для медиа и контента
- `Кайтен для редакций.docx` — издательства и редакции

### 💼 Для офисных функций
- `Кайтен для бухгалтерии.docx` — бухгалтерия и финансы
- `Кайтен для маркетинга.docx` — маркетинг и реклама
- `Кайтен для саппорт.docx` — техподдержка и customer support

### 🏢 Для специфических клиентов
- `Кайтен для веб-агентств.docx` — веб-агентства и IT-агентства
- `Госкомпании и учреждения.docx` — государственные организации

### 🌐 HTML примеры
- `kaiten-career.html` — лендинг для рекрутинга/карьеры
- `kaiten-crm-landing.html` — лендинг для CRM функции
- `kaiten-banner-v2.html` — банер/промо версия
- `kaiten-january.html` — сезонный лендинг

## 🎯 Как использовать примеры

1. **Найди релевантный пример** по сфере целевой аудитории
2. **Изучи структуру** — как раскладываются блоки
3. **Заметь тон текстов** — как пишет Екатерина про Kaiten в этой сфере
4. **Посмотри примеры кейсов** — какие компании упомянуты
5. **Поймай стиль** — короткие ли предложения, используются ли цифры и метрики

## 📊 Типичная структура в примерах

Почти все примеры следуют такой структуре:

1. **Заголовок** — конкретное решение для конкретной сферы
2. **Подзаголовок** — почему это важно именно для них
3. **Ключевые выгоды** — 3-5 главных преимуществ Kaiten для этой сферы
4. **Кейсы** — реальные примеры компаний (если есть)
5. **Детальное описание фич** — как конкретные возможности решают их задачи
6. **CTA** — призыв к действию (демо, пробный период, консультация)

## 💡 Что NOT делать на основе примеров

- ❌ Не копировать текст дословно
- ❌ Не использовать структуру, если она не подходит для новой сферы
- ❌ Не забывать адаптировать под новую аудиторию
- ✅ Использовать как вдохновение и эталон качества

---

**Помни:** Перед созданием нового лендинга обязательно откройи и изучи примеры из этой папки!
