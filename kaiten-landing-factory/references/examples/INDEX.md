# Примеры верстки лендингов

Эта папка содержит HTML примеры готовых лендингов для вдохновения и как эталон верстки.

## 🌐 HTML примеры верстки

- `kaiten-career.html` — лендинг для рекрутинга/карьеры (пример адаптивной верстки)
- `kaiten-crm-landing.html` — лендинг для CRM функции (пример сложной структуры)
- `kaiten-banner-v2.html` — банер/промо версия (пример компактной верстки)

## 📋 ТЗ примеры и эталоны

Для **содержания и структуры новых лендингов** используй эталоны из папки [`landing_tz_examples`](https://github.com/katyakaiten/katyakaiten/tree/main/kaiten-landing-factory/references/landing_tz_examples):

- `Кайтен для веб-агентств.md` — веб-агентства и IT-агентства
- `Кайтен для маркетинга.md` — маркетинг и реклама
- `Кайтен для редакций.md` — издательства и редакции
- `Посадочная страница для ИТ команды.md` — IT-команды, разработка
- `Посадочная страница для Строительство, инженеры.md` — строительство и инжиниринг
- `Управление портфелем проектов.md` — управление проектами, PM
- `Госкомпании и учреждения.md` — государственные организации

## 🎯 Как использовать примеры

### Для содержания (ТЗ):
1. Открой релевантный `.md` файл из `landing_tz_examples`
2. Изучи структуру блоков
3. Посмотри как написаны тексты (тон, метрики, примеры)
4. Скопируй эту структуру для своего ТЗ

### Для верстки (HTML):
1. Открой релевантный `.html` файл из этой папки
2. Посмотри как сделана адаптивность
3. Заметь как используются классы и компоненты
4. Используй этот стиль при верстке своего лендинга

## 📊 Типичная структура в эталонах

Все ТЗ в `landing_tz_examples` следуют этой структуре:

1. **Hero блок** — заголовок, подзаголовок, CTA
2. **Кейсы** — 3-5 реальных компаний с результатами
3. **Блоки решений** — по 3-5 болей, для каждой свой блок
4. **Гибкие блоки** — дополнительные возможности для этой сферы
5. **Миграция** — как быстро внедриться
6. **Онбординг** — пошагово за 30 минут
7. **Готовые шаблоны** — примеры для их сферы
8. **CTA** — финальные призывы к действию
9. **FAQ** — частые вопросы
